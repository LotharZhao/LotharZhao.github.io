Git：常用命令

发表于 2015-06-13 分类于 Git 本文字数： 194 阅读时长 ≈ 1 分钟

Git的命令较多，本篇记录了常用的几个命令，以备后续方便查看。

克隆/下载已有的远程仓库

$ git clone url (自定义本地仓库名字，可空)

初始化本地仓库

$ git init

查看当前工作区状态

$ git status (-s 简要)

将文件添加到暂存区域

$ git add (文件名或.)

提交

$ git commit (文件名或空) (-a 直接跳过添加暂存区) -m “提交的说明”

查看git提交的日志

$ git log (-数字 查看几条日志，不写查看全部)

显示提交和工作树等之间的更改

$ git diff

从另一个存储库或本地分支获取并集成

$ git pull

将本地分支的更新，推送到远程主机

$ git push origin master
