OpenStack简介

发表于 2017-04-04 分类于 OpenStack 本文字数： 495 阅读时长 ≈ 2 分钟

openstack概述

开源的云操作系统

Apache License 商业化最友好的开源协议

  • 计算 Compute Nova
  • 存储 Storing Cinder Swift Glance
  • 网络 Network Neutron
  • 管理界面 Horizon
  • 身份认证 Keystone

Nova 计算服务

  • 实例生命周期管理
  • 提供基于REST的API
  • Hypervisor屏蔽，支持KVM、Xen、Hyper-v、Docker e.g.
  • 支持大规模水平伸缩

Neutron 网络服务

  • 虚拟网络管理
  • 面向租户的网络管理（SDN）
  • 高度可配置的plug-in架构
  • 基于REST的API

Keystone 身份认证服务

  • 认证与授权
  • 用户、租户和角色管理
  • 服务目录管理
  • 支持多认证机制（MySQL、Ldap、AD e.g.）

Glance 镜像服务

  • 镜像注册、查询
  • 基于角色的访问控制
  • 支持多格式的镜像（raw、qcow2 e.g.）
  • 支持多种后端存储（S3、Swift、File system e.g.）

Horizon 控制面板服务

  • 用户自助管理界面

Cinder 块存储服务

  • 提供类似EBS的块存储
  • 减少Nova的复杂性，降低Nova的负载
  • 支持多种后端存储

Swift 对象存储服务

  • 可靠对象存储
  • 没有容量限制
  • 无单点故障
  • 支持S3的API

Heat 编排服务

  • 模板定义环境
  • 云资源拓扑与编排服务
  • 兼容AWS CloudFormation

Ceilometer 计量服务

  • 资源监控与度量

概念

  • User 用户
    • 用于身份验证，一个用户可以属于多个租户
  • Tatent 租户
    • 相当于用户组的概念，一个租户包含有多个用户，可以映射一个项目成员或者部门
  • Role 角色
    • 系统权限集合的概念，一般默认有admin和member
  • Instance 实例
    • 代表计算能力的单位，比如虚拟机、Linux Container
  • Volume 卷
    • 块存储，类似网络磁盘
  • Snapshot 快照
    • Instance或者Volume某个状态的备份
  • Virtual Router 虚拟路由
  • subnet 子网
  • Region
    • 侧重地理高可用
  • Available Zone
    • 小范围服务高可用
  • Cell
    • 二级调度
  • Host Aggregate
    • 物理兼容性隔离、同属性设备划分
