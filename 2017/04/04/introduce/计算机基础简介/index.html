计算机组成简介

发表于 2017-04-04 | 更新于: 2017-04-04 | 分类于 introduce

计算机设主要有五个部分组成：运算器、控制器、存储器、输入设备、输出设备。CPU是中央处理器，是控制和运算器；存储器的主要功能是存储程序和各种文件；输入输出设备一般统称I/O，用于连接计算机内外部。这些组件的连接则需要总线和主板来完成。

1.CPU

CPU全称Central Processing Unit中央处理器，是一台计算机的运算核心和控制核心。功能主要是解释计算机指令以及处理计算机软件中的数据，分为三个关键阶段：提取、解码和执行。CPU从内存或CPU缓存中取出指令，放入指令寄存器，并对指令译码。它把指令分解成一系列的微操作，然后发出各种控制命令，执行微操作系列，从而完成一条指令的执行。

1.1.组成

CPU主要由控制器、运算器、寄存器和时钟四部分组成。

  • 控制器：负责把内存上的指令、数据读入寄存器，并根据指令的结果控制计算机
  • 运算器：负责运算从内存中读入寄存器的数据，算数逻辑单元ALU
  • 时钟：负责发出CPU开始计时的时钟信号
  • 寄存器：是中央处理器内的组成部分。它们可以用来暂存指令、数据和地址。可以将其看作是内存的一种。根据种类的不同，一个CPU内部会有 20-100个寄存器。
种类 功能
程序计数器PC 用于存放下一条指令所在单元的地址
累加寄存器 存储运行的数据和运算后的数据
标志寄存器 用于反应处理器的状态和运算结果的某些特征以及控制指令的执行
指令寄存器 储存正在被运行的指令，CPU内部使用，程序员无法对该寄存器进行读写
栈寄存器 存储栈区域的起始位置
程序状态寄存器PSW 控制当前状态是内核态还是用户态
基址寄存器 存储数据内存的起始位置
变址寄存器 存储基址寄存器的相对地址
通用寄存器 存储任意数据
…

1.2.CPU指令执行过程

  1. 取指令：将内存中的指令读取到 CPU 中寄存器的过程，程序寄存器用于存储下一条指令所在的地址
  2. 指令译码：指令译码器按照预定的指令格式，对取回的指令进行拆分和解释，识别区分出不同的指令类别以及各种获取操作数的方法
  3. 执行指令：完成指令所规定的各种操作，具体实现指令的功能
  4. 访问取数：根据指令的需要，有可能需要从内存中提取数据，此阶段的任务是：根据指令地址码，得到操作数在主存中的地址，并从主存中读取该操作数用于运算
  5. 结果写回：Write Back，把执行指令阶段的运行结果数据写回到某种存储形式，结果数据经常被写到CPU的内部寄存器中，以便被后续的指令快速地存取

1.3.多核多线程

  • 线程、进程：进程包含线程，一个进程中包含多个线程。线程是CPU调度和分配的基本单位，进程是操作系统进行资源分配(CPU、内存、硬盘等)的最小单位

  • 单核CPU：实现多进程依靠于操作系统的进程调度算法，比如时间片轮转算法，有3个正在运行的程序（即三个进程），操作系统会让单核CPU轮流来运行这些进程,然后一个进程只运行2ms，这样看起来就像多个进程同时在运行,从而实现多进程

  • 多核CPU：一枚处理器集成了多个完成的计算引擎（内核），可以实现多线程的并行
  • 多核CPU和单核CPU对于进程来说都是并发的，不是并行

1.4.CPU频率

也就是CPU的时钟频率，CPU运算时的工作的频率，1秒内发生的同步脉冲数，振荡器两个脉冲之间的时间量，单位赫兹（Hz），决定了CPU的运行速度

1GHz=10^3MHz=10^6KHz= 10^9Hz

1.5.制程

晶体管门电路的尺寸，单位是纳米（nm），2019年发展到7纳米。

理论上，制程工艺越小，芯片面积越小，频率更高，功耗越小

1nm=0.001um=10^-6mm=10^-9m

摩尔定律：当价格不变时，集成电路上可容纳的元器件（晶体管）的数目，约每隔18-24个月便会增加一倍，性能也将提升一倍

2.内存

  1. 内存（Memory）是计算机中最重要的部件之一，它是程序与CPU进行沟通的桥梁。计算机中所有程序的运行都是在内存中进行的，因此内存对计算机的影响非常大，内存又被称为主存，其作用是存放CPU中的运算数据，以及与硬盘等外部存储设备交换的数据。只要计算机在运行中，CPU就会把需要运算的数据调到主存中进行运算，当运算完成后CPU再将结果传送出来，主存的运行也决定了计算机的稳定运行
  • 高速缓存Cache：分为一级缓存（L1 Cache）、二级缓存（L2 Cache）、三级缓存（L3 Cache）这些数据，它位于内存和CPU之间，是一个读写速度比内存更快的存储器。当CPU向内存写入数据时，这些数据也会被写入高速缓存中。当CPU需要读取数据时，会直接从高速缓存中直接读取，如需要的数据在Cache中没有，CPU会再去读取内存中的数据。
  • 随机存储器RAM：内存中最重要的一种，既可以从中读取数据，也可以写入数据。当机器断电后，内存中的信息会丢失
  • 只读存储器ROM：只能用于数据的读取，不能写入数据，但是当机器停电时，这些数据不会丢失。非常快，而且便宜。在计算机中，用于启动计算机的引导加载模块（bootstrap）就存放在ROM中，一些I/O卡也采用ROM处理底层设备控制

操作系统

1.基础

操作系统是指控制和管理整个计算机系统的硬件和软件资源，并合理的组织调度计算机的工作和资源的分配，以提供给用户和其他软件方便的接口和环境集合。是随着计算机研究和应用的发展逐步形成并发展起来的，是计算机系统中最基本的系统软件

基本特征

  • 并发：计算机系统中同时存在多个运行着的程序，因此它具有处理和调度多个程序同时执行的能力
  • 共享：资源共享，是指系统中的资源可供内存中多个并发执行的进程共同使用。有两种方式：互斥共享方式、同时访问方式。
  • 虚拟：把一个物理上的实体变为若干个逻辑上的对应物。利用多种虚拟技术，分别用来实现虚拟处理器、虚拟内存和虚拟外部设备。虚拟技术可归纳为：时分复用技术和空分复用技术
  • 异步：在多道程序环境下，允许多个程序并发执行，但由于资源有限，进程的执行不是一贯到底，而是走走停停，以不可预知的速度向前推进，这就是进程的异步性
