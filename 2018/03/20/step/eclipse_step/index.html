Eclipse安装配置

发表于 2018-03-20 分类于 部署 本文字数： 1k 阅读时长 ≈ 4 分钟

绝对没有人想到我会写一篇关于Eclipse的安装配置的文章，这最基础的开发软件应该在Java学习之初就学会的。可谁能知道，更新了公司的入网小助手之后电脑崩溃了，只能重装，可怜我Eclipse装C盘，又得重新弄，这以前日积月累对Eclipse的配置，一下子在改，太费事了。所以现在写这篇文章，以备以后可快速查阅，不用再到处百度了，另多图提醒。

Eclipse下载安装

下载

1、点击进入Eclipse官网，本次准备下载最新的OXYGEN(4.7.0)的版本，该版本较上一版本主要增加了对Java9的支持等，不多赘述，开始下载。

2、点击上图标注的Download Packages，进入不同版本选择的界面。这里选择Eclipse IDE for Java EE Developers的版本，一般项目涉及Web的最好选择该版本。

Eclipse常用版本
Eclipse IDE for JavaEE Developers：该版本集成了Java EE开发常用插件，方便动态web网站开发，适合Java web开发者使用。
Eclipse IDE for Java Developers：该版本集成CVS，Git，XML编辑器，Mylyn, Maven integration和WindowBuilder等插件，适合普通Java开发者。
Eclipse IDE for C/C++ Developers：该版本集成了良好的C/C++语言支持，适合C/C++开发者。
Eclipse IDE for Eclipse Committers：该版本是Eclipse最基础的版本，适合Java SE个人开发者或希望根据自己需求配置插件的开发者使用。
Eclipse IDE for Java and DSL Developers:该版本集成了Java和XTend ID等插件，适合Java和DSL开发者使用。

3、 如下图，先选择右侧Oxygen Packages，及Eclipse的Oxygen(4.7.0)版本；然后选择右侧Eclipse IDE for JavaEE Developers的Windows 64-bit位。

4、 这次进入的是最后的节点镜像服务器选择的界面，当然不选默认下载也行，这里我先选择Select Another Mirror，再选择入下图的中国科学技术大学的节点。

5、 点击上面的镜像后会跳转到下图界面，该页面不固定，静静等待几秒即可进入下载。

安装

这里我下载的是压缩包的版本（好像有安装版，没有细看），直接右键解压到指定目录即可。

Eclipse配置

修改Eclipse内存

在Eclipse安装目录下，用文本编辑器打开eclipse.ini文件，根据电脑配置情况修改最后的内存配置，下图为我修改后的配置。

修改字体大小

Window–>Preferences–>General–>Appearance–>Colors and Fonts–>Basic–>Text Font，选中后点击右侧edit，进入如下图页面，在大小的位置，根据自己的需求修改字体大小。我是直接在大小处修改为18。然后点确定应用即可。

安装SVN插件

因为公司的代码是用SVN托管的，Eclipse没有自带SVN，需要安装SVN的插件。Help–>Eclipse Marketplace，进入下图页面，在Find处输入svn搜索SVN插件。因为我已经安装SVN插件，所以第一项显示的是Installed。第一次安装点击第一项右下角的Install，后面根据提示下一步下一步即可。

配置tomcat

Window–>Preferences–>Server–>Runtime Environments，点击右侧Add选项，在弹出的对话空选择你的Apache的tomcat版本，next。Browse选择tomcat的位置，JRE使用默认即可，如下图。

Window–>Show View–>Servers，调出Servers标签，右键new–>Server，选择刚配置的tomcat，如下图。

双击刚配置出的tomcat名称，进入配置页面，如下图。修改tomcat的启动和停止时间，端口号，以及服务器路径，即编译的项目保存的路径，因为下图是我已发布项目，无法再编辑，我习惯选第一项，就保存再workspace的目录中。
